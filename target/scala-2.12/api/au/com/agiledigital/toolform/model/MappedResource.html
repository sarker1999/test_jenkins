MappedResource 

final case class MappedResource(path: String, tagName: String, environment: Map[String, String], ports: Seq[String], links: Seq[String], resource: Resource) extends ProjectElement with Product with Serializable

A MappedResource is a Resource that has been mapped by the developer and can be Composed into the .yml config.

path

the full path to the mapped resource config.

tagName

the name of the docker image used to provide the resource.

environment

the environment variables to pass to the docker image.

ports

the ports that will be exposed by the resource docker.

links

the links that will be made to the resource docker.

resource

the resource that was mapped.
