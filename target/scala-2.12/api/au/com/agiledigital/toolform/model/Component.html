Component 

Companion object Component

final case class Component(idOverride: Option[String], path: String, name: String, builder: String, settings: Option[Config], componentGroup: Option[ComponentGroup], environment: Map[String, String] = Map(), exposedPorts: List[PortMapping] = List(), externalPorts: List[PortMapping] = List()) extends ToolFormService with Product with Serializable

A Component is a project element that is built the project. It can be converted into a Docker image and composed into the .yml.

idOverride

the identifier of the component.

path

the full path to the component in the configuration.

name

the name of the component.

builder

the Docker image that will be used to build the component.

settings

the settings for building the component.

componentGroup

the group that this component is a part of (may be None).

environment

a map of environment variables to inject into the container.

exposedPorts

a list of ports that are exposed to other services in the internal network.

externalPorts

a list of ports that are exposed to an outside network. (This is usually done through the edge/nginx reverse proxy but sometimes you need to expose a container)
