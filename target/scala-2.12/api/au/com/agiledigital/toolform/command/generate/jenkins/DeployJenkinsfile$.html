  15.  def parsedBuilder(builder: String): String

    Parses a builder name, this is specified in the project.conf file for component, e.g.

    Parses a builder name, this is specified in the project.conf file for component, e.g. "public_api": { ... builder: "play25", ... } We may use this builder name to locate a certain Jenkins library, e.g. play25-jenkins-library. The name of the builder has to be specific and it should not be a docker image URI. There are some component definitions in project.conf using docker build as: "elastic_search": { ... builder: "docker.agiledigital.com.au:5000/agile/docker-build:201708061841", ... } But the Jenkins library prefix should be "docker" (docker-jenkins-library), so we make sure it returns the "docker" as builder name instead.

    builder

    from the project.conf to be parsed.

    returns

    parsed builder prefix.
